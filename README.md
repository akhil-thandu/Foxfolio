🦊 Foxfolio
Personal Portfolio Manager — tracks Indian stocks, US stocks/ETFs, and crypto in one unified dashboard.
Stack

Frontend: Next.js + TypeScript
Backend: FastAPI (Python)
Database: PostgreSQL
Cache: Redis

Quick Start

Start database and cache: docker-compose up -d
Start backend: cd backend && uvicorn app.main:app --reload
Start frontend: cd frontend && npm run dev
